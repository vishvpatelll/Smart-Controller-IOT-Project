# 🏠 Smart Controller (Home Automation System)

## 📌 Overview
The **Smart Controller** is a home automation system that allows users to control electrical appliances using a smartphone. It uses an Arduino UNO as the main controller and communicates with a mobile application via Bluetooth. Users can turn devices ON/OFF using a single tap or voice commands, making home control more convenient and efficient.

## 🚀 Features
- 📱 Control home appliances using a smartphone  
- 🔊 Voice command support  
- 🔵 Bluetooth-based wireless communication  
- ⚡ Real-time ON/OFF control  
- 👴 Helpful for elderly and physically challenged users  
- 💡 Energy-efficient usage by controlling unused devices  

## 🛠️ Technologies Used
### Hardware:
- Arduino UNO  
- Bluetooth Module (HC-05 / HC-06)  
- Relay Module  
- Electrical Appliances  

### Software:
- Arduino IDE (C/C++)  
- MIT App Inventor (Android App Development)  

## ⚙️ How It Works
1. The user sends a command via the mobile app (button or voice).
2. The command is transmitted using Bluetooth.
3. Arduino UNO receives and processes the command.
4. The relay module switches the appliance ON/OFF accordingly.
